# 03. 자주 쓰는 명령어

모두 백엔드 폴더(`stockbot/backend`)에서 실행.

## uv (패키지/환경)
```powershell
uv sync                 # 의존성 설치 + .venv 준비 (lock 기준)
uv add <패키지>          # 라이브러리 추가 (pyproject·lock·venv 자동 갱신)
uv remove <패키지>       # 라이브러리 제거
uv run <명령>            # .venv 안에서 명령 실행
uv run python           # 프로젝트 환경의 파이썬 실행
```

## 서버 실행
```powershell
uv run uvicorn stockbot.main:app --reload --port 8000
```
- `--reload`: 코드 바뀌면 자동 재시작 (개발용)

## DB 마이그레이션 (Alembic)
```powershell
uv run alembic revision --autogenerate -m "설명"   # 모델 변경 → 새 마이그레이션 생성
uv run alembic upgrade head                         # 최신까지 적용 (테이블 반영)
uv run alembic downgrade -1                         # 한 단계 되돌리기
uv run alembic current                              # 현재 적용된 버전 확인
```

## Postgres (Docker)
```powershell
docker compose up -d            # Postgres 띄우기 (백그라운드)
docker compose down             # 내리기 (데이터는 볼륨 유지)
docker compose logs -f db       # DB 로그 보기
docker ps                       # 떠있는 컨테이너 확인
```

## 테스트 / 코드 품질
```powershell
uv run pytest                   # 테스트 실행
uv run ruff check .             # 린트 (문제 찾기)
uv run ruff format .            # 포맷 (자동 정렬)
```
