# 00. 사전 준비물 설치 (PC에 1회)

빌드/실행 전에 **내 PC에 한 번만** 깔아두는 것들. (프로젝트 코드와 무관한 도구들)

## 1. uv (파이썬 패키지/환경 매니저)
가상환경·라이브러리·파이썬 버전을 알아서 관리해줌. **uvicorn(서버)과는 다른 것.**

Windows(PowerShell) 셋 중 하나:
```powershell
# (추천) 공식 설치기 — 파이썬 없어도 됨
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

# 또는 pip (이미 파이썬 있으면 간단)
pip install uv

# 또는 winget
winget install astral-sh.uv
```
확인:
```powershell
uv --version
```

## 2. Docker Desktop (Postgres 컨테이너용)
DB(Postgres)를 도커로 띄울 거라 필요.
- https://www.docker.com/products/docker-desktop 에서 설치 → **실행(켜두기)**
- 확인:
```powershell
docker --version
docker compose version
```

## 3. Python
- **따로 안 깔아도 됨** — uv가 프로젝트에 맞는 Python 3.12.13을 알아서 설치/관리.
- (이미 3.12.x 있으면 그대로 써도 OK)
