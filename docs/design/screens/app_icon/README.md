# 주식비서 앱 아이콘

토스 블루(#3182F6) 기반 앱 아이콘. 추천안은 **인사이트 차트**(상승 라인 + AI 스파크)예요.

## 파일
- `icon-insight-1024.png` — ★ 추천. **정사각 마스터**(1024×1024). iOS/스토어 제출용 (모서리는 OS가 자동으로 둥글게 처리하니 이 정사각 원본을 그대로 올리세요)
- `icon-chat-1024.png` — 대안 B. AI 말풍선 + 막대 차트
- `icon-candle-1024.png` — 대안 C. 캔들(정통 핀테크)
- `icon-ring-1024.png` — 대안 D. 포트폴리오 링 게이지
- `icon-insight-ios-rounded-1024.png` — 추천안의 둥근 모서리 미리보기(참고용, 제출엔 정사각 마스터 사용)

## 사용 가이드
- **iOS**: `icon-insight-1024.png`(정사각, 투명도 없음)를 App Store Connect / Xcode 1024pt 슬롯에 등록. 모서리 둥글림은 시스템이 처리.
- **Android**: 어댑티브 아이콘은 전경/배경 레이어를 권장해요. 단색 배경(#3182F6 그라데이션) + 흰 마크 구조라 분리가 쉬워요. 필요하면 전경(마크 only, 투명 배경)·배경 레이어를 따로 만들어 드릴게요.
- **다른 사이즈**(180/120/87/60/40 등)는 1024 마스터에서 축소 생성하면 돼요. 필요하면 세트로 뽑아 드릴게요.

## 편집
벡터 원본은 `AppIcon.dc.html`에 있어요(SVG, 1024 좌표계). 색/형태 수정 후 PNG 재추출 가능. `앱아이콘.dc.html`에서 4종 컨셉과 설치 미리보기를 한눈에 비교할 수 있어요.
