# Handoff: 토스 스타일 주식 비서 앱 (AI 포트폴리오 분석)

## Overview
토스증권 오픈 API + 자체 AI Agent를 결합한 "내 주식 비서" 모바일 앱입니다. 사용자가 앱 페어링 키와 토스증권 API 키를 연결하면, 보유 종목·평가손익을 한눈에 보고, AI 포트폴리오 분석/채팅/리포트를 받고, 실적·배당 캘린더를 확인하며, 디스코드로 알림을 받습니다. 전체 종목(코스피·코스닥·미국)을 탐색하며 관심종목을 등록할 수 있습니다.

## About the Design Files
이 번들의 `주식비서.dc.html` 파일은 **HTML로 만든 디자인 레퍼런스**입니다 — 의도한 룩앤필과 동작을 보여주는 프로토타입이며, 그대로 복사해 쓰는 프로덕션 코드가 아닙니다.

목표는 이 HTML 디자인을 **대상 코드베이스의 기존 환경(React Native, Flutter, SwiftUI, 네이티브 등)에서 그 환경의 확립된 패턴·라이브러리로 재구현**하는 것입니다. 아직 코드베이스가 없다면, 이 프로젝트(모바일 앱)에 가장 적합한 프레임워크를 선택해 구현하세요. 모바일 앱이므로 **React Native / Expo** 또는 **Flutter**, 또는 네이티브(SwiftUI/Compose)가 자연스러운 선택지입니다.

`.dc.html`은 디자인 도구의 컴포넌트 포맷입니다. `support.js`(런타임)와 `<x-dc>`/`<sc-if>`/`<sc-for>` 같은 태그는 프리뷰 전용이므로 **이식하지 마세요**. 마크업의 인라인 스타일 값(색상·간격·폰트·radius)과 로직 클래스(`class Component`)의 상태/계산 로직만 참고하면 됩니다.

> 참고: 이 프로토타입의 시세·잔고·AI 응답·페어링 검증·캘린더 일정은 모두 **데모용 더미 데이터**입니다. 실제로는 토스증권 오픈 API(OAuth 토큰 발급 → 잔고/시세 조회)와 자체 AI Agent 백엔드, 페어링 엔드포인트에 연결해야 합니다.

## Fidelity
**High-fidelity (hifi)** — 최종 색상·타이포그래피·간격·인터랙션이 확정된 픽셀 단위 목업입니다. 토스(Toss) 핀테크 스타일을 따릅니다: 화이트 카드 + 연한 그레이 배경 + 블루(#3182F6) 포인트 + Pretendard 굵은 타이포 + 둥근 모서리. 개발자는 코드베이스의 기존 라이브러리/패턴으로 이 UI를 픽셀에 가깝게 재현해야 합니다.

> 한국 시장 관례: 상승/이익 = **빨강(#F04452)**, 하락/손실 = **파랑(#3182F6)**. (서구권과 반대) 이 컨벤션을 반드시 유지하세요.

## 앱 구조 (Navigation)

- **온보딩 플로우** (탭바 없음): 인트로 → 1단계 앱 페어링 키 → 2단계 토스 API 키 → 연결 로딩 → 완료 → 홈
- **메인 앱** — 하단 4탭: `홈` / `탐색` / `관심` / `전체(설정)`
- **하위 화면** (탭바 위 push, 좌상단 `‹` 뒤로가기): 종목 상세, AI 분석, AI 리포트, 종목별 리포트, 실적·배당 캘린더
- **플로팅 버튼** "🤖 AI 비서": 홈/탐색/관심 화면에서 우하단에 떠 있고, 누르면 AI 분석 화면으로 이동
- 뒤로가기는 **스택(stack) 기반**입니다. 화면 진입 시 이전 화면을 push, `‹` 또는 완료 시 pop. 탭 전환은 스택을 비웁니다.

기기 프레임(검은 베젤 + 노치/다이나믹 아일랜드 + 상태바 9:41 + 홈 인디케이터)은 프로토타입 표현용입니다. 실제 앱에서는 OS가 제공하므로 **구현 불필요** — 내부 콘텐츠(width 390 기준)만 가져가세요.

## Interactions & Behavior

- **온보딩 게이팅**: 페어링 키(1단계) 성공 → API 키(2단계) → 연결 로딩 → 완료. 각 단계는 이전 단계 완료 없이 진행 불가
- **페어링 검증(데모)**: `페어링 확인` → 1.4초 `확인 중…` 스피너 → `TOSS-AI`로 시작하면 성공(✓, 0.65초 후 2단계 자동 진입), 아니면 에러 메시지. **실제 구현 시 백엔드 페어링 엔드포인트로 교체**
- **API 연결(데모)**: `연결하기` → 2.3초 로딩 → 완료. **실제는 토스 OAuth 토큰 발급으로 교체**
- **버튼 활성화**: 입력 글자 수(데모 4자) 충족 시에만 활성. 비활성은 회색 + cursor not-allowed + 핸들러 no-op
- **별 토글**: 탐색·상세·관심 어디서든 즉시 관심 목록(watch 배열) 추가/삭제, 모든 화면에 즉시 반영
- **검색/필터**: 입력·칩 변경 시 실시간 필터링
- **AI 채팅**: 추천 질문 탭 → 유저 버블 + AI 응답 버블 추가, `bubble .25s ease` 등장
- **캘린더**: 날짜 탭 → 선택일 변경 + 해당일 일정 표시. 월 화살표 → 이동(선택일 1일로 리셋). 필터 칩 → 실적/배당 표시 토글
- **뒤로가기**: 스택 기반 pop. 탭 전환은 스택 초기화
- **애니메이션**: 화면 진입 `scrIn .3s`(opacity+translateY 10px), 성공 체크 `pop .4s`(scale), 스피너 `spin .9s linear infinite`, 채팅 `bubble .25s`
- **알림 설정 의존성**: 마스터 푸시 off → 하위 3개 비활성(opacity 0.4) + 토글 no-op

## State Management
필요한 상태 변수:
- `screen` — 현재 화면 키 (`onboard`/`home`/`explore`/`detail`/`ai`/`reports`/`stockReport`/`calendar`/`watch`/`settings`)
