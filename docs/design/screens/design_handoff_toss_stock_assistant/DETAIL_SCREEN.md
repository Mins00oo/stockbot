# 종목 상세 화면 — 재설계 스펙 (Stock Detail)

토스증권 오픈 API로 **실제 조회 가능한 데이터**에 맞춰 재설계한 종목 상세 화면입니다. 한국/미국 종목을 분기 처리합니다. 이 문서는 개발자가 화면을 그대로 재현하고 API 필드에 바인딩할 수 있도록 작성됐어요.

> 프로토타입의 호가·체결·거래량·상하한가 등은 종목별로 **그럴듯하게 생성한 데모 값**입니다. 실제로는 토스 API 응답 필드로 교체하세요. 데모 생성 로직은 `주식비서.dc.html`의 `orderBookEl()` / `tradesEl()` / `priceChartEl()` / `volumeBarsEl()` / `limitStrs()` / `volStr()`에 있습니다.

| 현재가 | ✅ | ✅ |
| 등락률·등락액 | ✅ | ✅ |
| 원화 환산 가격 | — | ✅ (환율 적용) |
| 차트 — 기간별(1일·1주·1달·3달·1년) | ✅ | ✅ |
| 거래량 | ✅ | ✅ |
| 호가 10단계 | ✅ | ✅ |
| 최근 체결 | ✅ | ✅ |
| 상한가·하한가 | ✅ (±30%) | ✅ (참고 밴드) |
| 종목 기본정보(이름·거래소·산업·로고) | ✅ | ✅ |

## 2. 화면 구성 (위 → 아래)

### (1) 헤더
- `‹` 뒤로(32×32, 26px) · 종목명(15/700, 중앙) · ★/☆ 관심 토글(★ `#FFB400` / ☆ `#C4CCD4`)

### (2) 종목 헤더
- 로고 46×46 radius 14 (브랜드 컬러 bg + 흰 이니셜 18/800)
- `🇰🇷/🇺🇸 거래소 · 티커` 13/600 `#8B95A1` + 종목명 18/800 `#191F28`

### (3) 현재가 블록
- 현재가 32/800 letter-spacing −1px
- 등락 `▲/▼ N.NN%   ±금액` (상승 빨강 `#F04452` / 하락 파랑 `#3182F6`) 15/700
- **미국만**: 옆에 원화 환산 `₩...` 13/600 `#B0B8C1`

### (4) 거래량·상한가·하한가 스트립
- 흰 카드 radius 16, 3등분(세로 구분선). 각 칸: 라벨 11.5/600 `#8B95A1` + 값 14/700
- 거래량(`#191F28`) · 상한가(`#F04452`) · 하한가(`#3182F6`)
- 상/하한가 = 전일종가 기준, 한국 ±30%(호가단위 스냅), 미국은 참고 밴드(±10%)

### (5) 거래경고 배너 — 한국 전용, 경고 있을 때만
- 경고 종류별 색 카드(예: 단기과열 `#E8590C`/bg `#FFF0E6`, 투자주의 `#F08C00`/`#FFF9DB`, 투자위험 `#F03E3E`/`#FFE3E3`, VI `#1098AD`/`#E3FAFC`)
- `⚠️` + `거래경고 · {종류}` 13.5/800 + 설명 문구 12.5/`#4E5968`

### (6) 차트 카드
- 헤더: `차트` 14/800 + 우측 **기간 수익률 칩**(선택된 기간의 수익률, 예 `3달 +12.83%` / `1년 −15.14%`, 상승 빨강·하락 파랑, 13/700) — 기간을 바꾸면 함께 갱신
- 라인 차트 height 128 (인라인 SVG, 라인 + 하단 그라데이션, 상승 빨강/하락 파랑, `preserveAspectRatio:none` viewBox 330×110)
- 거래량 히스토그램 height 38 (막대, 상승 `#FAD4D8`/하락 `#CFE0FB`)
- **기간 선택 바**(차트 아래, 풀폭): `1일 · 1주 · 1달 · 3달 · 1년` 5개 버튼. 회색 트랙(bg `#F8F9FB` radius 12, padding 4) 위에 5등분 알약. 각 버튼 `flex:1` · padding 11px 0(큰 hit target) · radius 9 · 13px. **활성**: bg `#EAF2FE` + 텍스트 `#3182F6` + 700. **비활성**: 투명 bg + `#8B95A1` + 600. 기본 선택 `3달`. 탭하면 차트·거래량·수익률 칩이 그 기간으로 재계산
- 캡션: `1일·1주·1달·3달·1년 · 실시간 스트리밍은 제공되지 않아요` 11/`#B0B8C1` (가운데 정렬)
- 주봉·월봉은 만들지 않음

### (7) 호가 / 체결 카드 (탭)
- 헤더: 제목(`호가 10단계` 또는 `최근 체결`) + 세그먼트 토글 `호가` / `체결`
- **호가 탭**: 3열 그리드(매도잔량 | 호가 | 매수잔량). 위 10행=매도(ask, 잔량 바·숫자 파랑 `#3182F6`, bg `#E7F0FF`), 아래 10행=매수(bid, 빨강 `#F04452`, bg `#FFE9EC`). 가운데 호가 셀은 전일종가 대비 색(위 빨강/아래 파랑) + 등락% 소형 표기. 잔량 바 width ∝ 잔량/최댓값
- **체결 탭**: 3열(체결시각 | 체결가 | 체결량). 시각 내림차순, 체결가 전일종가 대비 색, 체결량 `▲/▼`로 매수/매도 표시
- 캡션: `10단계 호가 · 최근 체결 스냅샷` (실시간 아님)

### (8) 내 보유 현황 — 보유 종목만
- 흰 카드. 행: 보유 수량 / 평균 단가 / 평가금액 / 평가손익(`+금액 (+N.NN%)`) / 일간 손익 / 거래비용(추정)
- 손익은 상승 빨강 / 하락 파랑
- **미국**: 평가금액·평가손익·일간손익을 `$값` 메인 + `₩환산` 보조 라인으로 **병기**

### (9) 종목 정보
- 흰 카드. **미국**: 시가총액 · PER · PBR · EPS · 배당수익률 · 52주 최저~최고
- **한국**: 시가총액 · 52주 최저~최고 (PER·PBR·EPS·배당 **없음**)
- 하단: 회사 설명 문단 13.5/`#6B7684` line-height 1.6

### (10) 예정된 일정 — 캘린더 일정 있을 때만
- 📅 카드 + `YYYY년 M월 D일 · 실적발표/배당락` → 탭 시 실적·배당 캘린더로 이동

### (11) 하단 고정 액션바
- **`.scr` 스크롤 컨테이너 밖**, 폰 프레임(`position:relative`) 기준 `position:absolute; left:0; right:0; bottom:0; z-index:25`로 고정 → 스크롤과 무관하게 항상 화면 하단
- 별 토글(54×52 radius 14 bg `#F2F4F6`) + `매수하기`(flex:1 height 52 radius 14 bg `#3182F6` 흰 16/700)
- 상세 콘텐츠 컨테이너는 `padding-bottom:110px`로 액션바 가림 방지
- 네이티브 구현 시: 화면 하단 고정 바(예: SafeArea 하단 sticky/absolute)로 처리. 탭바가 없는 상세 화면이라 충돌 없음

## 3. 토스 API 매핑 가이드 (구현 시)

- **현재가/등락/거래량/상하한가**: 시세 조회(현재가) 응답 → (3)(4) 블록
- **차트**: 기간별 시세 조회 → (6). 기간 선택 바(1일·1주·1달·3달·1년)로 각 기간 엔드포인트(or range 파라미터) 전환. 기간 수익률 칩 = 해당 기간 시작가 대비 등락. 주봉·월봉은 만들지 말 것
- **호가**: 10단계 호가 조회 → (7) 호가 탭(매도/매수 잔량·가격)
- **체결**: 최근 체결 조회 → (7) 체결 탭. 스냅샷 폴링(실시간 웹소켓 아님)
- **종목정보/펀더멘털**: 종목 기본정보 + (미국) PER·PBR·EPS·배당 → (9). 한국은 펀더멘털 필드를 호출/표시하지 않음
- **거래경고**: 한국 종목의 시장경보(정리매매/투자주의·경고·위험/단기과열) + VI 상태 → (5)
- **보유 현황**: 계좌 잔고 조회(종목별 수량·평균단가·평가금액·평가손익·손익률·일간손익·수수료/세금) → (8). 미국은 환율로 원화 병기
- **예정 일정**: 실적·배당 캘린더 데이터 → (10)

## 4. 상태(state)
- `detailPeriod`: `'1일'` | `'1주'` | `'1달'` | `'3달'` | `'1년'` — 차트 기간 선택(기본 `'3달'`)
- `bookTab`: `'호가'` | `'체결'` — 호가/체결 탭
- `selectedId`: 현재 보는 종목 id
- 국가 분기: `country(s)` = market이 KOSPI/KOSDAQ면 `'KR'`, 그 외 `'US'`

## 5. 색상 (한국 시장 관례)
- 상승/이익 = **빨강 `#F04452`**, 하락/손실 = **파랑 `#3182F6`** (서구권과 반대 — 반드시 유지)
- 그 외 토큰은 메인 `README.md`의 Design Tokens 참조
